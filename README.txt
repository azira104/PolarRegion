# PolarBytes Repository

Welcome to the PolarBytes Repository project! This repository serves as a central hub for research data related to the polar regions, including Antarctica and the Arctic.


## Usage

Once the development server is running, you can access the website at `https//polar.umk.edu.my`. The website provides access to the data repository and allows users to explore and download data related to the polar regions.
